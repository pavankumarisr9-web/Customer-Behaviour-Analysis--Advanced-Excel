# Customer-Behaviour-Analysis--Advanced-Excel

This project analyzes customer behavior using Advanced Excel techniques to identify purchasing patterns, segment customers, and uncover key trends that influence sales performance. Using functions such as Pivot Tables, Lookup formulas, Conditional Formatting, and Data Visualization tools, the analysis highlights high-value customers, frequently purchased products, seasonality trends, and revenue drivers. The project enables data-driven insights into customer preferences, helping businesses optimize marketing strategies, improve product offerings, and enhance customer retention. The structured workflow includes data cleaning, transformation, exploratory analysis, segmentation, and results visualization, all aimed at supporting better business decisions through clear, actionable insights.
